# postcode
 
 httpリクエストを送ってAPIを叩く！
 
 Yahoo!ディベロッパーネットワークのYOLP(地図サービス)の郵便番号APIを叩いて住所を表示する。
 
# DEMO

```
values.Add("query", "049-3514")
```

上のように入力すると、、、

「北海道山越郡長万部町富野」(東京理科大学長万部キャンパスの所在地)

が得られました。一年生の時、荷物を送るときなど何回も書いた住所、、、、
 
# Requirement
 
Golang

Yahoo! Developer Networkに登録してappID(Client ID)を取得(無料)
